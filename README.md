# Premium Currency Converter

A real-time currency converter with live exchange rates, multi-currency comparison, and conversion history.

## Features

- Real-time exchange rates for 150+ currencies
- Multi-currency conversion and comparison
- Conversion history tracking
- Favorite currencies management
- Offline support
- Responsive design

## Getting Started

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```
